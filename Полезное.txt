JWT - это токен, используемый для цифровой подписи.
Работает похоже на RSA, только наоборот - приватным ключом
коодируются передаваемые данные, а раскодируются публичным
ключом. Смысл этого в том, чтобы никто не мог подделать 
информацию, пересылаемую между сервером и клиентом, так
как для этого нужен приватный ключ. Но данные не зашифрованы, 
прочитать их может кто угодно.

Записи (records) - это новый тип ссылочных типов, представленный в C# 9.0, который предоставляет встроенную функциональность для неизменяемых (immutable) объектов с поведением, основанном на значении.
Основные особенности records
	1.Неизменяемость по умолчанию - свойства init-only (могут быть установлены только при инициализации), но это только по умолчанию, если реализовывать record полностью, как класс, то в нее можно писть и не init свойства, а и set
	2.Сравнение по значению - автоматическая реализация Equals, GetHashCode и операторов ==, != (сравнивается по возвращаемому значению публичных свойств (не полей), кроме вычисляемых (у которых установлен только get))
	3.Метод ToString - автоматически форматирует вывод всех свойств
	4.Поддержка деконструкции - можно разложить запись на отдельные переменные
	5.Ключевое слово with - для создания копии с изменениями (non-destructive mutation)
